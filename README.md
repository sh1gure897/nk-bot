## nk-bot
＜概要＞
nk-botは、Discordサーバーを一新することを目的とした管理用BOTです。既存のVortex系BOTに近い挙動を持ち、サーバー構成を短時間で変更できます。サーバー運営やテスト環境の初期化を効率よく行いたい開発者向けのツールです。

＜主な機能＞

サーバー名変更：サーバー全体の名称を自動で変更します

チャンネル生成：指定した構成で複数のチャンネルを作成します

メッセージ送信：各チャンネルに定型メッセージを送信します

＜インストール方法＞
まず、GitHubリポジトリからソースコードを取得します。gitコマンドを使用してローカル環境にクローンしてください。Pythonで動作するため、事前にPythonの実行環境が必要です。

git clone https://github.com/sh1gure897/nk-bot/

＜使い方＞
インストール後、コマンドラインからmain.pyを実行します。起動後、Discord上で指定されたコマンドを入力することでBOTが動作します。コマンドはサーバーの管理権限を持つユーザーが実行してください。

python main.py

nuke

＜動作環境＞
Python 3.9以上がインストールされている環境で動作します。Discord BOTを動かせる一般的なOSであれば利用可能です。事前にDiscordのBOTトークン設定が必要です。
